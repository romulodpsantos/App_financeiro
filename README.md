# 💰 Meu Controle Financeiro

Sistema completo de controle financeiro pessoal com todas as funcionalidades implementadas.

## 🚀 Funcionalidades

### ✅ Regras Implementadas

1. **Gastos iniciam como pendentes** - Só entram nos gastos do mês quando marcados como pagos
2. **Controle de pessoas atualizado automaticamente** - Com informações de parcelamento
3. **Gastos recorrentes atualizam "A receber"** - Na home
4. **Sistema de flag para pagamentos** - Só cria ganhos quando confirmado
5. **Previsão para próximos meses** - Cálculo automático de parcelas futuras
6. **Pendentes a pagar** - Atualização automática na home
7. **Previsão para pessoas** - Informações sobre pagamentos futuros

### 🎯 Funcionalidades Principais

- 📊 **Dashboard completo** com resumo financeiro
- 💳 **Controle de transações** com filtros avançados
- 🔄 **Gastos recorrentes e parcelados**
- 📈 **Relatórios e gráficos** detalhados
- 👥 **Controle com pessoas** e previsões
- 📱 **Design responsivo** para mobile e desktop

## 🛠️ Como Usar

1. **Baixe todos os arquivos** para uma pasta
2. **Abra o `index.html`** no navegador
3. **Comece a usar** - não precisa de instalação!
